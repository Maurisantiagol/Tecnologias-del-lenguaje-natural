# Tilin´s kitchen - chatbot

## Descripción

El proyecto consiste en un chatbot para un restaurante italiano llamado "Tilin´s kitchen". El chatbot es capaz de:

- Reservar mesas
- Consultar el menú
- Obtener recomendaciones de platillos
- Consultar ingredientes
- Cancelar o modificar reservas

## Requisitos

- Python 3.6 o superior
- Flask
- spaCy
- joblib
- pandas
- numpy
- matplotlib
- kagglehub

## Instalación

```bash
pip install -r requirements.txt
```

## Ejecución

```bash
python chatbot.py
```
